# PathoScope 安装说明

1. 系统要求：macOS 14+，Apple Silicon。
2. 把 `PathoScope.app` 拖到旁边的 `Applications`。
3. 第一次运行若系统拦截，请在 Finder 中右键 PathoScope，选择“打开”。
4. MRXS 可直接读取；SVS / NDPI / SCN / TIFF 请先在终端执行 `brew install openslide`。

当前 Preview 使用 ad-hoc 签名，尚未 Apple notarization；只从本仓库 Release 下载，并核对同目录 SHA-256。

研究/学习用途，非医疗器械，不用于临床诊断。
